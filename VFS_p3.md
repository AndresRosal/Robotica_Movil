El sistema se basa en un campo de fuerzas para mover el robot hacia un objetivo evitando obstáculos. Se usan dos fuerzas: una atractiva, que apunta hacia el objetivo, y otra repulsiva, que se obtiene a partir de los datos del láser y apunta en sentido contrario a los obstáculos. Cada una se multiplica por un factor que cambia según la situación. Si hay obstáculos cerca, los factores se reducen para que el movimiento sea más suave; si no los hay, se aumentan para avanzar con más decisión.

La fuerza total se calcula promediando las componentes de los vectores atractivos y repulsivos. Con la dirección de esa fuerza se obtiene el error angular del robot, y de ahí la velocidad angular mediante un controlador PD. La velocidad lineal se toma del módulo del vector resultante y se limita entre valores máximos y mínimos para mantener un movimiento estable.

Si la componente del vector indica que el objetivo está detrás, el robot avanza despacio y gira según el controlador hasta orientarse. Cuando la distancia al objetivo es menor de una unidad se considera alcanzado y pasa al siguiente. Si se detiene por completo, gira ligeramente siempre hacia el mismo lado hasta salir de esa situación. Por último, se dibujan las fuerzas a escala y se ejecutan las velocidades calculadas.
